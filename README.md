# Автосервис "Северный"

**Десктопное приложение для записи клиентов в автосервис на PyQt6.**

---

## О проекте

Приложение предназначено для автоматизации записи клиентов в автосервис. Оно позволяет:

- **Регистрироваться** новым пользователям с подтверждением email
- **Входить** в личный кабинет по email и паролю
- **Добавлять информацию** об автомобилях (марка, модель, год выпуска, пробег)
- **Выбирать дату и время** для записи на сервис
- **Просматривать историю** всех своих автомобилей
- **Фильтровать** историю по году выпуска и пробегу
- **Экспортировать** отфильтрованные данные в Excel

---

## Технологии

Проект построен на современном стеке:

| Компонент | Технология |
|-----------|------------|
| Интерфейс | PyQt6 |
| База данных | SQLite |
| ORM | SQLAlchemy |
| Хеширование | bcrypt |
| Экспорт в Excel | OpenPyXL |
| Отправка email | SMTP (smtplib) |
